Buttons

<x-buttons> is used to groups several related buttons together.

When "tracking" attribute is set, it will also automatically toggle buttons on/off when clicked.

Examples

Tracking "-1" (default)

Does not toggle automatically any buttons.

Tracking "0"

Toggles the clicked button on/off and other buttons off.

Tracking "1"

Toggles the clicked button on and other buttons off.

Tracking "2"

Toggles the clicked button on/off.

Tracking "3"

Toggles the clicked button on/off, but toggles off only if there is at least one other button toggled on.
