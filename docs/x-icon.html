Icon

<x-icon> is used to display a specific icon from a symbol-based SVG iconset file.

Xel comes with several predefined iconsets containing the most often used icons:

  • Material - located in /node_modules/xel/icons/material.svg
  • Material Outlined - located in /node_modules/xel/icons/material-outlined.svg
  • Fluent - located in /node_modules/xel/icons/fluent.svg
  • Fluent Outlined - located in /node_modules/xel/icons/fluent-outlined.svg

It is recommended that you create your own iconset file that contains only the icons needed by your app. This could be done e.g. by copying a predefined iconset file and then manipulating it with Boxy SVG.
