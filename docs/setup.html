Setup

1 Install Xel

Run from the console:

npm install xel

Note: Xel NPM package has no dependencies and it does not execute any scripts during installation.

2 Link Xel

Add to the <head>:

<script src="node_modules/xel/xel.js" type="module"></script>

6 Set locale (optional)

Add to the <head> to use English Polish locale:

Note: You can skip this step if your aren't planning to translate your app UI into multiple languages. Otherwise you should provide a path to your own FTL locale file. The file name should consist from ISO 639 language code, optionally followed by "-" and ISO 3166 territory code, e.g. en.ftl, en-US.ftl or en-GB.ftl.

Note: You can also provide multiple paths separated by commas. If a message is not found in the first locale file, Xel will look for it in the subsequent locale files.
